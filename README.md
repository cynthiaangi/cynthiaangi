## 👩‍💻 Sobre mim

Depois de mais de cinco anos atuando na área de Engenharia Química, encontrei no desenvolvimento de software uma nova forma de aplicar meu perfil analítico e resolver problemas.

Hoje atuo como **Desenvolvedora Front-end**, participando da evolução de aplicações web e mobile utilizadas por milhares de clientes. Trabalho diariamente com **Angular, AngularJS, TypeScript e JavaScript**, desenvolvendo novas funcionalidades, integrando APIs REST e colaborando com equipes ágeis para entregar soluções de qualidade.

Além da experiência profissional, desenvolvo projetos Full Stack utilizando **React, Java, Spring Boot, MySQL e AWS**, buscando evoluir continuamente em Engenharia de Software e desenvolvimento de aplicações escaláveis.

Também utilizo ferramentas de IA como **Cursor**, **Claude Code** e **ChatGPT** para apoiar o desenvolvimento, revisão de código e aumento da produtividade.
